# Tulip Automate

Build automated PPTX slide reports by agent based on databases in BigQuery and PostgreSQL.

---

## Graph Generation Architecture

We have migrated from standalone, hardcoded Python scripts to a centralized, JSON-driven graph generation architecture. All graphs are now generated by a single agent skill (`graph-maker`) that reads dynamic data payloads.

The legacy standalone scripts (`scripts/generate_line_friends_graph.py` and `scripts/generate_traffic_sources_graph.py`) are deprecated and no longer used.

### The Agent Skill
*   **Skill:** `graph-maker`
*   **Script:** `.gemini/skills/graph-maker/scripts/make_graph.py`
*   **Command:** 
    ```bash
    python3 .gemini/skills/graph-maker/scripts/make_graph.py "$(cat payloads/<payload_name>.json)"
    ```

### Available Graphs and Mappings

Currently, we support 5 distinct graphs for the monthly report. When generated, they should be archived into their respective month folders (e.g., `Graph_March/`).

#### 1. Line Friends Growth
*   **Data Payload:** `payloads/graph_line_friends.json`
*   **Output Name:** `line_friends_report_graph.png`
*   **Description:** A line chart tracking Total Friends, Target Reach, and Blocked Count with MoM percentages.

#### 2. Traffic Sources
*   **Data Payload:** `payloads/graph_payload.json`
*   **Output Name:** `traffic_sources_graph.png`
*   **Description:** A line chart showing where new friends came from (Ads, URLs, Other).

#### 3. Registers vs. Collecting Points
*   **Data Payload:** `payloads/graph_registers_collect_points.json`
*   **Output Name:** `registers_vs_collecting_points_graph.png`
*   **Description:** A bar chart comparing cumulative registered users against the number of active monthly point collectors.

#### 4. Redeem (By Items)
*   **Data Payload:** `payloads/graph_redeem_items.json`
*   **Output Name:** `redeem_by_items_graph.png`
*   **Description:** A bar chart showing the total number of physical reward items claimed in a given month.

#### 5. Redeem (By Points)
*   **Data Payload:** `payloads/graph_redeem_points.json`
*   **Output Name:** `redeem_by_points_graph.png`
*   **Description:** A bar chart showing the total number of *points burned* on each reward type in a given month.

### Styling & Fonts
The graphs automatically apply the "Tulip" brand color scheme (Cocoa Brown `#5C3225` and Orange `#E97132`). The `make_graph.py` script automatically loads the **TH Sarabun New** font from the `fonts/` directory to ensure perfect rendering of Thai characters.
